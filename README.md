# Weather App

## Description

The Weather App is a simple web application that provides users with real-time weather information for multiple locations. It is built using basic HTML, CSS, and JavaScript, integrated with a weather API to fetch and display current weather conditions.

## Features

- Real-time weather updates for multiple cities.
- Temperature displayed in both Celsius and Fahrenheit.
- "Feels like" temperature taking into account wind and humidity.
- Current wind speed in kilometers per hour (Kph).
- Last updated timestamp for accurate data.
- Responsive design for accessibility across devices.
- User-friendly interface with minimalist design.

## Technology Stack

- HTML
- CSS
- JavaScript
- Weather API (https://www.weatherapi.com/)
